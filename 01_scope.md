\# Scope



\## Context (fictional)

Medium-sized organization with:

\- hybrid identity (cloud + on-prem)

\- Windows endpoints

\- file shares + business apps

\- remote access (VPN)

\- some public-facing services

\- multiple SaaS suppliers



\## In scope

\- Identity \& access

\- Email / collaboration

\- Endpoints

\- Servers \& backups

\- Network / remote access

\- Third parties

\- Detection \& incident response readiness



\## Out of scope (for this version)

\- OT/ICS

\- physical security

\- detailed pentest findings



\## Assumptions

\- limited security budget

\- small IT team

\- need for auditability and basic governance
