\# Cyber Risk Register (Portfolio)



This repository contains a practical cyber risk register for a fictional medium-sized organization.



\## Contents

\- `01\_scope.md` — scope \& assumptions

\- `02\_method.md` — scoring method (Likelihood x Impact)

\- `assets\_inventory.csv` — minimal asset inventory

\- `risk\_register.csv` — risk register (controls + treatment plan + owners + deadlines)

\- `03\_executive\_summary.md` — top risks and prioritized actions



\## How to use

Open `risk\_register.csv` in Excel/Google Sheets and filter by:

\- `level` (Critical/High)

\- `owner`

\- `target\_date`
